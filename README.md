WAR Terminal Game

Este é um jogo inspirado no clássico jogo de estratégia "WAR". Ele foi desenvolvido em C e roda no terminal. O objetivo é cadastrar territórios e definir a quantidade de tropas em cada um.

Como usar

Faça o download do arquivo game.c.

Certifique-se de ter um compilador C instalado, como o GCC.

Compile o jogo com o comando:

gcc game.c -o war

Execute o jogo no terminal:

./war
Como funciona

Ao iniciar o jogo, você será solicitado a cadastrar 5 territórios.

Para cada território, digite o nome, a cor e a quantidade de tropas (máximo 50).

Após cadastrar todos os territórios, o programa exibirá a lista com os territórios e suas tropas.

Requisitos

Compilador C (como GCC ou Clang)

Sistema compatível com terminal
